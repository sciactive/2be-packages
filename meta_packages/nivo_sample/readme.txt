To use this sample slider for an inline module, insert something like this:

[com_nivoslider/slider file="/pines/media/nivo_sample/images.txt" theme="default" width="400px" height="300px" pauseOnHover="true" pauseTime="10000" /]

Try out different themes. These are included:

default
none (No controls. Captions are placed directly over the whole slider.)
orman (Pictures must be 568px by 268px.)
pascal (Pictures must be 630px by 235px.)
